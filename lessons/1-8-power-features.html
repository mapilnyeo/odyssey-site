Module 1을 마무리합니다. 지금까지 배운 모든 스킬을 정리하고, 앞으로 더 배울 수 있는 강력한 고급 기능들을 미리 봅니다. 그리고 Module 2: Vibe Coding을 소개합니다.

당신이 마스터한 것들

스킬 하는 일 언제 쓰나요
파일 탐색 낯선 환경 빠르게 파악 뭔가를 물려받았을 때
@파일 → 추출 단일 문서에서 정보 뽑기 한 문서를 깊이 분석할 때
@폴더 → 종합 여러 파일에서 패턴 발견 비슷한 파일이 많을 때
템플릿 적용 일관된 형식의 출력 제어 표준화된 문서가 필요할 때
이미지 분석 시각 자료에서 인사이트 추출 스크린샷, 차트, 사진 있을 때
웹 리서치 현재 정보를 즉시 가져오기 최신 정보나 경쟁사 분석
서브에이전트 🧪 병렬 처리로 시간 절약 비슷한 것이 여러 개일 때
커스텀 에이전트 다양한 관점 동시에 획득 여러 이해관계자의 시각 필요
GEMINI.md 프로젝트 컨텍스트 영구 저장 세션이 바뀌어도 기억 유지
승인 모드 안전성과 속도 균형 리스크 수준에 맞게 조절
/rewind ・ /resume 실수 취소, 세션 복귀 뭔가 잘못됐을 때
Chain-of-Thought 더 깊은 분석 요청 복잡한 판단이 필요할 때
Policy Engine AI 행동 규칙 세밀 제어 보안·자동화 설정 필요 시

다음에 배울 고급 기능

🔌
MCP 연동
Slack, Notion, GitHub, DB 등 외부 서비스를 Gemini CLI로 직접 읽고 씁니다.
geminicli.com/docs → MCP Setup
💾
Checkpointing
Gemini CLI가 자동으로 세션 스냅샷을 저장합니다. 문제 발생 시 이전 지점으로 복원 가능.
geminicli.com/docs → Checkpointing
📦
Sandboxing
격리된 환경에서 Gemini를 실행합니다. 경계 밖의 파일은 절대 건드리지 않습니다.
geminicli.com/docs → Sandbox
🌐
Remote Subagents 🧪
웹에서 실행 중인 전문 AI 에이전트에 연결합니다. Agent2Agent 프로토콜 기반.
geminicli.com/docs → Remote Agents
