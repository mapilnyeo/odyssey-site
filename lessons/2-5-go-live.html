Go Live — 인터넷에 출시 🚀

내 컴퓨터에서만 보이던 퀴즈를 전 세계가 볼 수 있는 진짜 URL로 만드는 순간입니다. Vercel CLI 한 줄로 배포하고, 스마트폰에서 접속해보고, 친구에게 링크를 보내세요.

터미널에서 시작:
gemini> /start-2-5

Vercel이란?

Vercel은 Next.js 앱을 가장 쉽게 배포할 수 있는 플랫폼입니다. 무료 플랜으로 충분합니다.

⚡
초고속 배포
명령어 하나로 30초 안에 배포 완료.
🆓
무료 플랜
개인 프로젝트는 Hobby 플랜으로 충분합니다.
🔗
자동 URL 생성
배포 즉시 xxx.vercel.app 형태의 URL이 생성됩니다.
🔄
GitHub 연동
GitHub에 push하면 자동으로 새 버전이 배포됩니다.

배포 단계

1
Vercel 계정 생성
vercel.com/signup — GitHub 계정으로 가입하면 연동이 쉽습니다.
2
Vercel CLI 설치 및 로그인
$ npm install -g vercel
$ vercel login
3
배포!
$ vercel --prod

✓ Deployed to production.
🔗 https://coffee-quiz-xxxxx.vercel.app
🎉
URL 공유!
스마트폰에서 접속해보세요. 친구에게 링크를 보내보세요. 당신이 진짜 앱을 만들었습니다.
