Introduction

코스 오리엔테이션을 합니다. Gemini CLI가 무엇인지, Basecamp Coffee 시나리오가 무엇인지 알아봅니다. 이 레슨 이후 모든 학습은 실제 업무 상황에서 Gemini CLI를 사용하는 방식으로 진행됩니다.

터미널에서 시작:
gemini> /start-1-1

이 레슨에서 배우는 것

  • Gemini CLI가 ChatGPT·웹 AI와 무엇이 다른지 이해한다
  • Basecamp Coffee 시나리오와 나의 역할(신임 매니저)을 파악한다
  • STOP / USER / ACTION 패턴을 익혀 레슨 흐름을 따라간다
  • 공식 문서 geminicli.com/docs가 어디 있는지 알고 찾아볼 수 있다
  • Gemini CLI를 실행하고 첫 번째 명령어를 입력한다

Basecamp Coffee 시나리오

이 코스 전체는 Basecamp Coffee 라는 가상의 커피 체인 이야기로 전개됩니다. 당신은 방금 이 회사의 신임 매니저로 입사했습니다.

🏕️
Basecamp Coffee
태평양 북서부 45개 지점을 보유한 지역 커피 체인. 공동체, 진정성, 지역 아티스트를 중심 가치로 삼습니다.
📉
실패한 로열티 프로그램
$40K를 투자한 Basecamp Rewards가 기대에 미치지 못합니다. 가입자는 있지만 실제 참여율이 너무 낮습니다.
⏰
3개월의 시간
경영진이 기한을 설정했습니다. 3개월 내에 프로그램을 살리지 못하면 폐기됩니다.
💡
당신의 무기: Gemini CLI
단독으로 싸우지 않아도 됩니다. Gemini CLI로 데이터를 분석하고, 솔루션을 설계하고, 앱까지 만들 수 있습니다.

레슨이 진행되는 방식

모든 레슨은 터미널 안에서 Gemini CLI와 대화하며 진행됩니다. 세 가지 마커를 기억하세요:

STOP
멈추는 지점
Gemini가 여기서 기다립니다. 지시에 따라 직접 해보거나, 생각해보거나, 답해야 합니다.
USER
당신이 할 일
이 표시 뒤에 나오는 것이 당신이 터미널에 입력하거나 수행해야 할 내용입니다.
ACTION
Gemini가 할 일
Gemini가 내부적으로 수행하는 작업입니다. 파일을 읽거나, 분석하거나, 만드는 등의 행동입니다.

지금 시작하려면

# 1. Gemini CLI 설치 (이미 설치되어 있으면 건너뛰기)
$ npm install -g @google/gemini-cli

# 2. 코스 폴더로 이동
$ cd complete-course

# 3. Gemini CLI 시작
$ gemini

# 4. 레슨 시작
gemini> /start-1-1
💡 공식 문서: geminicli.com/docs/get-started에서 설치 가이드를 확인하세요.
