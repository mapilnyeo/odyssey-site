Build & Iterate

드디어 만드는 시간입니다. Gemini CLI가 Next.js 앱을 처음부터 빌드합니다. 완성되면 브라우저에서 확인하고, 스크린샷으로 피드백해서 원하는 모습이 될 때까지 반복합니다.

터미널에서 시작:
gemini> /start-2-3

Step 1 · Next.js 프로젝트 생성

# quiz-project 폴더에서 실행
$ npx create-next-app@latest . --typescript --tailwind --eslint --app --yes

이 명령어 하나로 완전한 Next.js 프로젝트 구조가 생성됩니다. Gemini가 자동으로 실행해줍니다.

Step 2 · 퀴즈 앱 빌드

Plan 모드를 활용한 2단계 접근:

Plan 확인
Shift+Tab으로 Plan 모드 진입
Gemini가 무엇을 만들지 계획을 먼저 보여줍니다. 동의하면 진행, 수정이 필요하면 피드백합니다.
Build 실행
일반 모드로 전환 후 빌드 요청
gemini> "REQUIREMENTS.md를 기반으로 퀴즈 앱을 만들어줘"
Gemini가 파일들을 생성합니다. 지켜보기만 하면 됩니다.

Step 3 · 로컬 실행 및 확인

# dev 서버 시작
$ npm run dev

▲ Next.js 14.x
- Local: http://localhost:3000
✓ Ready in 2.1s

브라우저에서 http://localhost:3000 을 열면 퀴즈가 보입니다.

⚠️ localhost:3000은 내 컴퓨터에서만 접속 가능합니다. 인터넷 공개는 2.5 Go Live에서 합니다.

Step 4 · Iteration Loop

첫 버전은 완벽하지 않아도 됩니다. 이것이 실제 제품 개발 방식입니다:

🔄 Iteration Loop
📸 스크린샷 찍기 → 📋 Ctrl+V 붙여넣기 → 💬 변경사항 설명 → 🔄 새로고침 → 😊 반복
💡 맥 사용자는 Ctrl+V(Cmd+V 아님)로 스크린샷을 붙여넣습니다. Gemini가 이미지를 직접 보고 변경합니다.
