GitHub — 코드를 클라우드에 저장

내 컴퓨터에만 있는 코드는 언제든 사라질 수 있습니다. GitHub에 올리면 클라우드에 백업되고, 나중에 팀원과 협업하거나 Vercel로 배포할 수도 있습니다.

터미널에서 시작:
gemini> /start-2-4

3가지 핵심 개념

📸
Commit
코드의 스냅샷을 찍는 것. "지금 이 버전을 저장!"이라고 선언하는 행위입니다.
☁️
Push
로컬 commit을 GitHub 서버로 전송. 이것이 실제 클라우드 백업이 되는 순간입니다.
📁
Repository
프로젝트 코드가 저장되는 GitHub의 폴더. 변경 이력이 모두 기록됩니다.

GitHub CLI로 한 번에 설정

1
GitHub CLI 설치 및 로그인
$ winget install GitHub.cli # Windows
$ gh auth login
2
Repository 생성 및 코드 업로드
$ git init
$ git add .
$ git commit -m "Initial quiz app"
$ gh repo create coffee-quiz --public --source=. --push
3
GitHub에서 코드 확인
브라우저에서 github.com/[내 계정]/coffee-quiz에 접속하면 코드가 업로드된 것을 볼 수 있습니다.
💡 Gemini가 이 모든 명령어를 대신 실행해줍니다. 직접 외울 필요 없습니다.

왜 GitHub가 필요한가요?

이유 설명
☁️ 클라우드 백업 컴퓨터가 고장나도 코드는 안전
