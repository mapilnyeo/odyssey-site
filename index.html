바로 시작하기

딱 3단계면 됩니다

Gemini CLI만 설치하면 바로 시작할 수 있어요. 어렵지 않아요!

Step 1 · 설치

$ npm install -g @google/gemini-cli

Step 2 · 코스 폴더로 이동 후 실행

$ gemini

Step 3 · 첫 레슨 시작

gemini> /start-1-1
모듈 1

파일과 함께 일하기 — AI 기초 다지기

파일을 탐색하고, 분석하고, 패턴을 찾아냅니다. 어지럽게 인계받은 Basecamp Coffee 파일들을 정리하면서 Gemini CLI의 핵심 기능을 자연스럽게 익힙니다.

📂

모듈 1 · 기초 다지기

8개 레슨  ·  약 3시간  ·  코딩 경험 없어도 OK

레슨 1.1

코스 소개 — 지금 바로 시작

Gemini CLI가 뭔지, 이 코스가 어떻게 진행되는지 알아봅니다. Basecamp Coffee 매니저로 첫 출근! 보는 것이 아닌, 직접 하면서 배우는 방식에 익숙해집니다.

/start-1-1
LESSON 1.2

File Exploration & Visualization

상속받은 혼란스러운 파일들 탐색. Cursor 에디터 설정으로 파일을 눈으로 보면서 작업하는 환경 구성.

/start-1-2
레슨 1.3

@ 기호로 파일 다루기

@파일명만 입력하면 Gemini가 읽어냅니다. 파일 추출, 폴더 분석, 템플릿 적용, 이미지 분석, 웹 리서치 — 5가지 실전 시나리오.

/start-1-3
LESSON 1.4

Commands & Navigation

슬래시 명령어, 커스텀 TOML 명령, Esc 중단 · Esc×2 리와인드, 승인 모드, Chain-of-Thought, /memory 명령어, Policy Engine 소개.

/start-1-4
레슨 1.5

AI 팀 운영 — 병렬로 일 시키기 🧪 실험적

서브에이전트로 여러 파일을 병렬 처리. 경쟁사 분석을 동시에 처리하는 방법. 언제 병렬 처리가 유리한지 파악.

/start-1-5
레슨 1.6

나만의 AI 팀원 만들기

바리스타, 임원, 디자이너 — 역할이 다른 AI 팀원을 직접 만들어봅니다. 같은 문서도 관점에 따라 전혀 다른 피드백을 받을 수 있어요.

/start-1-6
레슨 1.7

GEMINI.md — Gemini의 장기 기억

매번 처음부터 설명하기 지치셨나요? GEMINI.md에 프로젝트 정보를 저장하면 Gemini가 켤 때마다 자동으로 다 알고 시작합니다.

/start-1-7
레슨 1.8

고급 기능 맛보기 & 모듈 1 총정리

지금까지 배운 것을 한눈에 정리합니다. MCP 연동, 체크포인팅, 샌드박싱 같은 고급 기능을 가볍게 소개하고 모듈 2로 넘어갑니다.

/start-1-8
모듈 2

직접 만들기 — 코드 없이 앱 출시

기획 → 만들기 → 다듬기 → 저장 → 배포. 코드 한 줄 없이 실제로 작동하는 웹앱을 인터넷에 올리는 경험을 합니다.

🚀

모듈 2 · 바이브 코딩

5개 레슨  ·  약 2시간  ·  Next.js · GitHub · Vercel 사용

레슨 2.1

준비 — 무엇을 만들지 정하기

여러분이 기획자, Gemini가 개발자가 됩니다. 기획→만들기→다듬기→저장→배포 흐름을 이해하고 프로젝트 폴더를 만들어 시작을 준비합니다.

/start-2-1
레슨 2.2

기획 — 만들 것을 구체적으로 정하기

커피 성격 퀴즈의 질문, 성격 유형, 커피 페어링, 디자인 방향을 결정합니다. 명확하게 요구사항을 적어야 Gemini가 정확하게 만들어줍니다.

/start-2-2
LESSON 2.3

Build & Iterate

Next.js 스캐폴드, 퀴즈 앱 빌드, localhost:3000 실행. 스크린샷 붙여넣기 → 반복 수정. 내 quiz 완성.

/start-2-3
레슨 2.4

GitHub — 코드를 클라우드에 저장

노트북이 고장나도 코드가 사라지지 않도록 GitHub에 백업합니다. 코드의 Google Drive라고 생각하면 됩니다. 배포를 위한 필수 단계예요.

/start-2-4
레슨 2.5

인터넷에 올리기 — 진짜 출시 🚀

명령어 한 줄로 실제 URL이 생깁니다. 스마트폰으로 접속하고 친구에게 링크를 보내보세요. 코딩 없이 첫 웹앱을 세상에 내놓는 순간입니다.

/start-2-5
핵심 기능

이 코스에서 배우는 Gemini CLI 기능

공식 문서 기반으로 검증된 Gemini CLI의 진짜 기능들

📝

GEMINI.md

프로젝트 메모리. 글로벌→프로젝트→폴더 계층 구조로 컨텍스트 영구 저장.

⚡

나만의 단축 명령어

자주 사용하는 작업을 .toml 파일로 저장해 /내명령어 형태로 불러씁니다.

🤖

나만의 AI 팀원 🧪

역할과 말투가 다른 AI 팀원을 마크다운 파일 하나로 직접 만들 수 있습니다.

🔄

안전 모드 전환

Shift+Tab으로 매번 확인받기 → 자동 실행 → 읽기 전용 탐색 모드를 오갑니다.

🧠

깊이 생각하게 만들기

"단계별로 생각해줘"라고 말하거나 /plan 모드를 쓰면 훨씬 꼼꼼한 답변을 받습니다.

🧲

/memory

/memory show · add · refresh로 프로젝트 컨텍스트 관리.

↩️

/rewind & /resume

Esc×2 또는 /rewind로 대화 되돌리기. /resume으로 이전 세션 복귀.

🛡️

AI 행동 규칙 설정

"이건 항상 허용", "이건 항상 물어봐"처럼 Gemini의 행동을 세밀하게 제어할 수 있습니다.

💾

자동 저장 & 복원

작업 중간 상태를 자동으로 저장합니다. 실수가 생겨도 언제든 이전 시점으로 되돌아갑니다.

🔌

MCP 연동

Slack, Notion, GitHub 등 외부 서비스를 Gemini CLI로 직접 제어.

📦

격리 실행 환경

Gemini를 완전히 분리된 공간에서 실행합니다. 위험한 실험도 안전하게 해볼 수 있습니다.

🌐

원격 AI 에이전트 연결 🧪

인터넷에서 실행 중인 전문 AI 에이전트에 연결해 협업합니다. AI 팀워크의 미래.
